# Event Tower

Event Tower is a professional event tracking application for Android that helps you visualize your timeline using a unique radar-style interface called "The Horizon".

## Features

- **Upcoming & Past Events**: Categorized lists with priority highlighting (Today, 10 Days).
- **The Horizon**: A pictorial radar map of your upcoming schedule with zoom and pan support.
- **Local Persistence**: Data is stored securely on your device using Room.
- **Import/Export**: Backup and restore your events using JSON format.
- **Privacy First**: Completely ad-free, local storage, and no unnecessary permissions.
- **Material 3 Design**: Modern, dark-themed UI with fluid animations.
